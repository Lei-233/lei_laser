# ⚡ PWM 激光功率控制测试指南

## 🚨 安全警告

### 激光危险！测试前必读

| 危险等级 | 说明 |
|---------|------|
| **高功率激光** | 可能造成永久性视力损伤或失明 |
| **反射光** | 间接反射也可能伤害眼睛 |
| **皮肤灼伤** | 高功率可能灼伤皮肤 |
| **火灾风险** | 聚焦光束可能引燃易燃物 |

### ✅ 必须遵守的安全规则

1. **始终佩戴激光防护眼镜**（对应激光波长）
2. **测试区域禁止无关人员进入**
3. **首次测试使用低功率**（5-10%）
4. **确保激光出射口朝向安全方向**
5. **熟悉紧急停止操作**（Ctrl+C）

---

## 🎯 快速验证（5分钟）

### 步骤1：基础控制测试

```bash
# 1. 加载驱动（如果还未加载）
insmod pwm_laser.ko

# 2. 确认初始关闭
./pwm_test off

# 3. 低功率测试（5%）
./pwm_test on 50
# ✅ 应该看到微弱激光。按Ctrl+C停止

# 4. 中等功率测试（30%）
./pwm_test on 300
# ✅ 激光明显变亮。按Ctrl+C停止
```

### 步骤2：动态渐变测试

```bash
# 渐变测试（0% → 50% → 0%）
./pwm_test fade 0 500 10
# ✅ 激光应该平滑渐变，按Ctrl+C停止并自动关闭
```

## 🚀 深度测试流程

### 1. 步进功率验证
```bash
# 5% → 50%，步长5%，每步停留1秒
./pwm_test step 50 500 50 1000
```
**观察重点：** 亮度随功率逐步增加，档位稳定且线性。

### 2. 脉冲闪烁测试
```bash
# 30%功率，100ms开/100ms关（5Hz闪烁）
./pwm_test pulse 300 100 100
```
**用途：** 验证 PWM 开关速度和激光器响应频率。

### 3. 系统联调（振镜联动）
```bash
# 终端1：启动振镜圆形扫描
./galvo_app 1

# 终端2：开启激光
./pwm_test on 500
# 预期：在目标面上看到激光圆形轨迹
```

## 🔧 常见问题与排查

| 问题 | 检查步骤/原因 |
|------|----------|
| **激光不亮** | 检查电源、TTL连接、驱动是否加载（`lsmod | grep pwm_laser`） |
| **功率不可调** | 激光器可能在恒功率模式，需切换到 TTL 调制模式 |
| **信号不稳定** | 检查电源质量、屏蔽线连接或 PWM 频率匹配度 |

## 📊 硬件参数备忘

- **PWM 频率**: 1 KHz (1ms周期)
- **分辨率**: 1000档 (0.1%步进)
- **电平**: 3.3V TTL
- **驱动节点**: `/dev/pwm_laser`
