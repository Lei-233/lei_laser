# 串口控制器使用指南 (Grbl 兼容)

## ⚡ 快速参考卡

### 常用 G 代码命令
| 命令 | 功能 | 示例 |
|------|------|------|
| `G0 X# Y#` | 快速移动 | `G0 X10 Y20` |
| `G1 X# Y# F#` | 直线运动 | `G1 X30 Y40 F1000` |
| `M3 S#` | 开激光 | `M3 S500` (50%功率) |
| `M5` | 关激光 | `M5` |
| `G90`/`G91` | 绝对/相对坐标 | `G90` |
| `?` | 实时状态查询 | `?` |
| `$I` | 版本信息查询 | `$I` |

### 串口配置参数
- **设备**: `/dev/ttyS3`
- **波特率**: `115200`
- **属性**: 8N1

---

## 🚀 启动与部署

### 1. 部署驱动与程序
```bash
# 加载必要的内核模块
insmod galvo.ko
insmod pwm_laser.ko

# 运行控制器
./uart_controller
```

### 2. 通信测试 (echo 方式)
```bash
echo "G0 X10 Y10" > /dev/ttyS3   # 移动到 (10,10)
echo "M3 S300" > /dev/ttyS3      # 激光 30%
echo "M5" > /dev/ttyS3           # 关闭激光
```

---

## 📖 详细功能说明

### 运动参数
- **X/Y 坐标**: 单位毫米 (mm)。1mm 对应 100 DAC 单位（可通过 `BEILV` 修改）。
- **F 速度**: 单位 mm/min。默认 G1 速度 10000，G0 速度 20000。
- **插补精度**: 默认 0.1mm 步进。

### 激光功率 (S 参数)
- **范围**: 0-1000 (S0=0%, S1000=100%)。
- **频率**: 1 KHz PWM。

## 🖥️ 上位机软件配置

### LaserGRBL / LightBurn
1. **协议**: Grbl 1.1
2. **波特率**: 115200
3. **功率范围**: 0 - 1000
4. **工作区域**: 根据实际振镜范围自定义。

---

## ⚠️ 安全与故障排查

- **紧急停止**: 按 `Ctrl+C` 停止软件，或物理断开激光电源。
- **权限问题**: 如无法打开串口，执行 `chmod 666 /dev/ttyS3`。
- **振镜不动**: 确认 `galvo.ko` 已加载且 `/dev/galvo` 节点正常。
