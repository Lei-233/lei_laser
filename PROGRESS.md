   - 交叉编译：ARM64，gcc-linaro-6.3.1
   - 依赖：galvo_hal.o + pwm_laser_hal.o

**技术要点:**

**精确计时原理:**
```c
// 旧方案（会闪烁）：相对延迟，累积误差
usleep(100);  // 每次100us，但实际延迟100us+函数开销

// 新方案（稳定）：绝对时间戳，消除误差
sleep_us_abs(100);  // 每次精确对齐到10kHz时钟
```

- 使用`CLOCK_MONOTONIC`保证时间连续性
- `TIMER_ABSTIME`标志确保对齐到绝对时间点
- 静态`next`时间戳记录下次唤醒点，自动校正累积误差
- 适用场景：高频率、长时间运行的实时控制（振镜/激光扫描）
